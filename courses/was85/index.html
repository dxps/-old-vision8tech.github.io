Virtual Machine setup

Use the following steps to create the VM into VirtualBox:

  • Start the VirtualBox application.
  • Click New for creating the virtual machine.
  • Use the following settings:
    • Name: was85host
    • Type: Linux
    • Version: RedHat 64bit
    • Memory size: 3080 MB (~3 GB)
    • Hard disk: Use an existing virtual hard disk file
